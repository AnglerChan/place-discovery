# place-discovery

一个给 Codex 用的城市探索 skill。帮你在一座城里挖出真正在运转的地方——厂房边上、港口外围、职业学院、物流节点、河堤、老电脑城、半停业的百货大楼之类。

不是旅游攻略。不排热门景点，不关心哪里出片。关心的是城市的产业、交通、水系、教育实际怎么运转，然后写成能读的地点文章。

## 怎么用

依赖高德 POI 做底层校验，网页搜索结果做补充证据。API key 放在环境变量或文件里就行：

```bash
export AMAP_API_KEY="你的key"

# 查行政区划
python3 scripts/amap_client.py district 无锡 --extensions all
# 关键词搜
python3 scripts/amap_client.py search 物联网 --city 无锡 --limit 20
# 查POI详情
python3 scripts/amap_client.py detail B0XXXX
```

脚本会自动从环境变量 `AMAP_API_KEY` 或者 skill 目录下的 `.amap_api_key` 读 key。文件已加进 `.gitignore`，不用怕漏提交。

没有高德 key 也能跑，skill 会用网页公开信息做判断，但不会瞎编 POI ID 或坐标。

## 看什么

分了四大类：

**教育与文化** — 大学、职校、技校、纺织馆、船舶馆、酱文化馆、气象馆、民间小博物馆

**产业与技术** — 工业园、工厂外围、研发中心、企业展厅、电动车、光伏、物联网、纺织、食品

**城市系统** — 港口、码头、泵站、水闸、变电站、热电、公交总站、冷链仓库、货运铁路、水厂

**低商业探索** — 湿地河堤、老商场、电脑城、家电卖场、半停业商业体、拆迁边界、围垦

## 不看什么

- 西湖/武康路/洪崖洞这类成熟观光符号
- 红色景点、革命纪念馆、烈士陵园、名人故居
- 大型综合博物馆（行业类小馆留，战争革命类不留）
- 文创街区、夜游、灯光秀、观景台、仿古街
- 翻墙/锁闭建筑/施工区/铁路高速穿越/敏感设施核心

## 写出来长什么样

最终是一篇中文杂志式文章，不是表格，不是打分表，不是 POI 列表。

内容写的是现场能看见的东西：门岗在哪个路口、围墙包到什么程度、货车从哪条路进厂、旧柜台还有多少、桥下空间怎么被用着、堤岸上写了什么编号。每个段落绑着一个具体地点，对着路名和片区写。坐标和 POI ID 只在内部分析阶段用，不出现在最终文章里。

安全提示直接写进段落，比如"厂区内部不要进入，沿公共道路看就行"，而不是一个单独的警告标签。

## 安全底线

只推荐公共道路、开放步道、市场/校园公共区域、官方展厅、有预约机制的场所。不教人翻墙、撬锁、穿铁路、靠近生产核心。现场以管理标识为准。
